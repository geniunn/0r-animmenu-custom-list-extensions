If you were previously using RP Emotes or another emote menu, and you now want to use RP Emote animations with 0r-animationmenu, please follow the steps below.

First download files and please follow these steps:

1. Open the 0r-animmenu folder in your resources directory.

2. Navigate to the customs folder inside it.

3. Locate the existing 10 language files in this folder.

4. Extract the downloaded ZIP file.

5. Copy the 10 language files from the ZIP and paste them into the customs folder.

6. When prompted, confirm that you want to replace the existing files.

7. Go back to the main 0r-animmenu folder and open the stream folder.

8. Delete the existing stream folder inside 0r-animmenu.

9. Copy the stream folder from the downloaded ZIP file and paste it into the 0r-animmenu folder.

Make sure all files are fully replaced before restarting your server.
